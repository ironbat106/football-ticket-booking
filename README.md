# Football Ticket Booking System

This repository contains the SQL script for the Football Ticket Booking System assignment.

## File Included

* `QUERY.sql`

## Description

The `QUERY.sql` file includes:

* Database table creation for `Users`, `Matches`, and `Bookings`
* Sample data insertion
* Required SQL queries from Query 1 to Query 7
* Primary key and foreign key relationships

## DBMS Used

PostgreSQL

## SQL Client Used

Beekeeper Studio
